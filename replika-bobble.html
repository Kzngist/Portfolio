About

Description: A ground up recreation of classic bust-a-move game Puzzle Bobble
Team Size: Solo
Development Cycle: ~8 weeks (Mar 2025 - Apr 2025)
Tools Used: Unity(game), Blender(model, UI), GarageBand(audio), ibisPaint(art)
Platforms: iOS, MacOS, PC

Features

Level Painter

  • A handy tool that simplifies the level creation process.
  • Seamlessly reads and edits existing levels.
  • Directly writes level data to the project assets folder.
  • Stores level data as ScriptableObjects, allowing fine-tuning through the inspector.

Multiplatform Support

  • Supports both mobile and desktop devices.
  • Supports mouse/keyboard/gamepad/touchscreen inputs, without conflict with combined usage
  • Automatic control scheme switches with Input System

Debug Systems

Debug Console

  • Runtime tool for modifying game behaviours and accessing additional features.
  • Displays custom messages and errors.
  • Works in builds for easy troubleshooting.

World & Screen Debugger

  • Visualizes object states and movements for easy debugging.
  • Displays custom messages on screen.

Optimized Game Structure

  • A singular game scene for all levels. Loads level data packed in ScriptableObjects.
  • Mid-game saving & loading system that accurately writes and reads game states.
  • Objects belong to "bobble trays" and calculations are performed in tray space. As such it opens possibility for creating dynamic gameplay, such as displacing or deforming bobble trays.
  • Utilizes a custom-made Hexagonal Coordinates System, which simplifies caculations, such as hexagonal rings around a point.
  • Collision detections are performed by manually spherecasting. This avoids potential glitches caused by the internal physics engine, also makes bobble movement align perfectly with the aim guide.
  • Modular and elegant design pattern and future proof.

Tweened Animations

  • All animations in game (UI interactions, gameplay, effects) are animated with tweening.
  • Elimates potential performance overhead caused by traditional animator components.

Visuals

  • Handcrafted UI elements and 3D models.
  • HDR support for applicable devices.
  • Fine-tuned colour palette and post processing profile.

SFX Design

  • Recorded with real world objects, such as collision of beer bottles.
  • Fine-tuned for immersive gaming experience.
